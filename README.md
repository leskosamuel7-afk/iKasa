# Domácí finance – jednoduché MVP

Jednoduchá webová aplikace inspirovaná rodinnými budget aplikacemi jako Wallet, Spendee a YNAB.

## Co umí

- přidávat příjmy a výdaje
- členové domácnosti
- vlastní kategorie
- měsíční rozpočty podle kategorií
- plánované platby
- přehled za zvolený měsíc
- ukládání dat do localStorage
- funguje bez backendu

## Jak spustit

Stačí otevřít `index.html` v prohlížeči.

## Jak dát na GitHub

1. Vytvoř nový repozitář na GitHubu
2. Nahraj soubory `index.html`, `styles.css`, `app.js`, `README.md`
3. V repozitáři zapni **Settings → Pages**
4. Jako source nastav branch `main` a root `/`
5. Appka poběží přes GitHub Pages

## Další krok

Můžeme přidat:

- přihlášení uživatelů
- Firebase / Supabase databázi
- sdílení mezi více zařízeními
- grafy
- export do CSV
- opakované platby
- tmavý / světlý režim
